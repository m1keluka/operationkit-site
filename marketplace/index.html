Installing an agent

Download a file. Hand it to Jarvis. Done.

Agents install by conversation, not by redeploy. Adding one is three steps and takes under a minute.

1

Download the agent.

Pick an agent above and click Download. You get a single .md file — the agent's full persona, rules, and skills. Open it and read it if you like; it's yours.

2

Open Jarvis and add it.

In your OperationKit, open Jarvis and say "add this agent", then provide the file you downloaded. Jarvis reads the persona and registers it in your workspace.

3

Select it and go.

The agent becomes selectable immediately — no redeploy, no restart. Route a task to it like any other agent, on whatever model you choose.

Because agents are just files, this works both ways.

Everything you download here you can also edit and re-add, and any agent you build yourself installs the exact same way. Nothing is trapped in a vendor's format — your agents move with you.
